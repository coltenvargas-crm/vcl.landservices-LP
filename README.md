# VCL Land Services — Landing Page

Live site: `https://vcl-land-services.netlify.app` (once deployed)

## Stack
- Pure HTML/CSS (single file)
- Hosted on Netlify
- Auto-deploys from this GitHub repo on every push to `main`

## Files
- `index.html` — The landing page (this is what gets published)
- `GO-LIVE-CHECKLIST.md` — First-time deployment setup
- `EDIT-WORKFLOW.md` — How ongoing edits flow from local → live
- `.gitignore` — Files Git should ignore

## Making Edits
See `EDIT-WORKFLOW.md`. Short version:
1. Edit `index.html`
2. Commit and push
3. Netlify auto-deploys in ~30 seconds

## Offer Packages Featured
1. **Land Revival** — Reclaim overgrown land
2. **Pond Revival** — Restore ponds to health
3. **Site Prep Transformation** — Prep land for building/development
